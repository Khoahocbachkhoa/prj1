# Pharmacy app
Ứng dụng quản lý tiệm thuốc đơn giản với các chức năng chính: 
    + quản lý kho
    + quản lý hóa đơn
    + quản lý khách hàng
    + thống kê báo cáo

Công nghệ sử dụng:
    FE: React + Vite
    BE: Flask + SQLite + Sqlalchemy
